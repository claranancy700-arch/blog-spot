# Blog Spot

This repository contains a basic React-based blog starter using Vite as the build tool.

## Getting Started

Make sure you have Node.js installed (v18+ recommended). Then run:

```bash
npm install
npm run dev
```

The development server will start on http://localhost:5173 by default.

## Project Structure

- `public/` – static assets and the root HTML file.
- `src/` – React components, styles, and entry point.
- `vite.config.js` – Vite configuration with React plugin.
